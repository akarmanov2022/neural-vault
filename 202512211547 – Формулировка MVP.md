# Формулировка MVP

## Для кого этот проект?

```
Люди, которые хотят развить навык слепой печати и совмещать это занятие с чтением литературы (классика/нон-фикшн)
```

## В чем ключевая ценность?

```
Тренировка слепой печати на осмысленном тексте с сюжетом, интересной мыслью, цитатой.
```

## Что входит в MVP?

- выбор произведения
- отображение текста
- переход к следующему фрагменту
- ввод текста пользователем
- подсчет
	- wpm
	- точность
