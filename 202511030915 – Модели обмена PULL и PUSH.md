# Модели обмена PULL и PUSH

## Модель PUSH
PUSH - модель асинхронной принудительной передачи данных
- данные принудительно передаются от производителя к подтребителю

## Модель PULL
- данные от производителя передаются по запросу от потребителя в установленном объеме
- имеет ряд проблем, связанных с затратами времени на ожидание ответа
